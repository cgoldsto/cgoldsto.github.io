Cards

Description and Use Details

Cards can be used on any page that displays multiple content items of the same type (projects, blog posts, etc.). Cards make content stand out by providing an associated image, title, and description.

Accessbility

  • Ensure sufficient space between cards, so that users can clearly read and process each card.
  • Use an h tag for the title in the card code. This will differentiate the title and description for screen reader use.
