Footer

Description and Use Details

The footer must be used on all Design by Cora Goldston webpages. The footer includes a copyright notice and names Cora Goldston as the copyright holder\.

Accessbility

  • Ensure that the text is large enough to be readable.
  • Keep the copyright date current, so that screen readers will read the correct year if someone is using screen reader assistance.

Code

/* Footer - Desktop */
position: absolute;
width: 1440px;
height: 52px;
left: 0px;
top: 111px;

/* Copyright Info - Desktop */

left: 36.46%;
right: 36.88%;
