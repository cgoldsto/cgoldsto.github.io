Navigation Bar

Description and Use Details

The navigation bar must be used on all Design by Cora Goldston webpages. On most pages, this will be at the top of the screen. The navigation bar allows the user to go from their current page to any other page on the website.

Accessbility

  • Ensure that the link colors contrast with the navigation bar's background color.
  • Ensure visual contrast with dark mode, in case users have dark mode enabled in their browsers.
